# Documento de Análisis
## Proyecto Final Unidad 1 - Moto Lavado SuperBiker

---

## 1. Descripción del Proyecto

Este documento presenta el análisis del sitio web desarrollado para **Moto Lavado SuperBiker**, un negocio de limpieza premium para motocicletas. El proyecto es un prototipo de interfaz web responsive desarrollado como trabajo colaborativo de la Unidad 1, siguiendo las especificaciones de la actividad y la plantilla proporcionada.

**Eslogan:** Limpieza premium para tu moto.

## 2. Objetivos

- Crear un sitio web funcional para Moto Lavado SuperBiker
- Implementar diseño responsive (celular, tablet y escritorio) con Bootstrap
- Utilizar elementos multimedia: imágenes, videos, formularios, tablas y listas
- Separar estilos en archivos CSS para personalización (colores, fuentes, tamaños)
- Documentar el proceso de desarrollo

## 3. Estructura del Proyecto

```
ProyectoFinalUnidad1/
```

## 4. Páginas Implementadas

### 4.1 Página de Inicio (index.html)

- **Navbar:** Logo del local, enlaces a Inicio y Servicios (Página 2)
- **Carrusel:** Mínimo 3 imágenes del servicio con leyendas y botones CTA
- **Secciones de contenido:**
  1. Tres columnas: texto (¿Por qué elegirnos?) | imagen | texto (Nuestra misión)
  2. Contenido centrado: bienvenida y botón
  3. Tres columnas: lista de servicios | horarios | tabla de precios
  4. Dos columnas: texto del proceso | placeholder de video
  5. Formulario de solicitud de información
- **Footer:** Información del creador, redes sociales (no reales), enlaces de interés

### 4.2 Página Servicios (pagina.html)

- **Navbar:** Igual a la página de inicio
- **Secciones de contenido:**
  1. Headline con texto y dos imágenes en grid
  2. Tabla de catálogo de servicios y precios + lista de beneficios
  3. Formulario de reserva de cita + placeholder de video
- **Footer:** Idéntico al de la página de inicio

## 5. Tecnologías Utilizadas

- **HTML5:** Estructura semántica (Celaya Luna, 2014)
- **Bootstrap 5.3:** Framework CSS, grid system, componentes (Navbar, Carousel, Cards, Tablas, Formularios), diseño responsive
- **Bootstrap Icons:** Iconos para redes sociales
- **CSS3:** Variables, gradientes, tipografía Montserrat, paleta de colores SuperBiker
- **Google Fonts:** Fuente Montserrat

## 6. Diseño y Branding

- **Paleta de colores:** Negro (#000000), Cyan (#00AEEF), Azul (#0054A6), blanco
- **Tipografía:** Montserrat (sans-serif moderna)
- **Estilo:** Limpio, tecnológico, orientado al cuidado de motos

## 7. Responsive Design

El sitio utiliza el sistema de grillas de Bootstrap para adaptarse a:

- **Celular:** Columnas apiladas, navbar colapsable, carrusel adaptado
- **Tablet:** Distribución intermedia
- **Escritorio:** Columnas en fila, máximo ancho del contenido

## 8. Bibliografía

- Celaya Luna, A. (2014). Creación de páginas web: HTML 5. Editorial ICB. (Pág. 14-18)
- Documentación Bootstrap 5.3: https://getbootstrap.com/docs/5.3/

## 9. Conclusiones

El prototipo cumple con todos los requisitos de la actividad: estructura de carpetas, páginas con navbar, carrusel, secciones de contenido multimedia, footer en tres columnas y hojas de estilos CSS para personalización. El diseño está alineado con la identidad visual de Moto Lavado SuperBiker y sirve como base para futuras iteraciones del sitio.
